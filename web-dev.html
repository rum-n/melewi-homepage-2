Questions ü§î

  • What is the difference between web development and web design?

    Web design consists of UX & UI design - showing you how the web or mobile app will look like, without actually being functional. Web development is using a programming language or a framework to make a functioning platform out of the created designs.
  • Do you develop everything in-house?
    We do some of the programming in-house and some of it with our partner company. It depends on the type of product you want to create and the technology we need to use.
  • What technologies do you use?
    On the backend we like to use Ruby / Ruby on Rails or Python or C# with React in the frontend. We also develop native apps for Android and iOS. For less technologically demanding projects, we like to use Wordpress.
  • What is the web development process?
    Just like our design process, we work in 2-week sprints, splitting every feature into individual tasks that are then assigned to different team members. Once a task is complete it goes through quality assurance before it's accepted as finished.
  • How long does it take to develop a website?
    That really depends on the complexity of the website. A website that has no backend functionality (e.g. no user login) can take us 2-3 weeks to develop after the design phase is complete. On the other hand, a more complex platform with a rich backend might take months to develop properly.
